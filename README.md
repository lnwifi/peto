# Club Patitas

Donde cada patita importa 🐾

## Setup

1. Clonar el repositorio
2. Copiar `.env.example` a `.env` y completar credenciales
3. `npm install`
4. `npm run dev`

## Tech Stack

- React 18 + Vite + TypeScript
- Tailwind CSS v4
- Supabase (Auth + Database + Storage + Realtime)
- Mercado Pago (Payments)
- React Router v6
